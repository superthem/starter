# 项目脚本说明

## 脚本功能

1. `create_application.sh`：用于创建应用程序项目模板。
2. `create_service.sh`：用于创建服务项目模板。

## 使用方法

### 1. 运行脚本

先进入你想要生成项目的目录，把对应文件拷贝过去，在终端中执行以下命令：

```bash
./create_application.sh  # 创建应用程序项目
./create_service.sh      # 创建服务项目
```

### 2. 输入信息

脚本会提示您输入以下信息：
- **公司名称**：纯英文小写字母，不允许用 `-` 连接。
- **项目名称**：纯英文小写字母，不允许用 `-` 连接。

### 3. 临时文件处理

脚本执行完成后，会询问是否删除克隆的临时文件（固定目录）。
- 输入 `y` 或 `Y` 删除临时文件。
- 输入其他内容或直接回车保留临时文件。

## 注意事项

1. 确保您的系统已安装 `git` 和 `bash`。
2. 脚本默认克隆的代码仓库地址为 `http://home.magicvector.cn:30080/common/`。
3. 临时文件默认保存在系统的 `Downloads` 目录下。

## 示例

```bash
$ ./create_application.sh
请输入公司名称（纯英文小写字母，不允许用-连接）: mycompany
请输入项目名称（纯英文小写字母，不允许用-连接）: myapp
是否删除临时文件（克隆的固定目录）？[y/N]: y
脚本执行完成！项目已创建为 myapp-application。
临时文件已删除。
```
